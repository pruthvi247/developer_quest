> You can quickly find all places where a particular class, method or variable is used in the whole project by positioning the caret at the symbol's name or at its usage in code and pressing ⌥ F7 (option+f7)

> To navigate to the declaration of a class, method or variable used somewhere in the code, position the caret at the usage and press ⌘ B (command+B).

> You can quickly navigate in the currently edited file with ⌘ F12 (Navigate | File Structure).
It shows the list of members of the current class. Select an element you want to navigate to and press the Enter key or the ⌘ ↓ key.
